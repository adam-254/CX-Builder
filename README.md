# Resume & Cover Letter Builder

A modern, React-based resume and cover letter builder with multiple professional templates.

## Setup

```bash
npm install
npm run dev
```

## Project Structure

```
src/
├── components/       # Reusable UI components
│   ├── Header.jsx
│   ├── Sidebar.jsx
│   └── Preview.jsx
├── pages/           # Page components
│   └── Builder.jsx
├── templates/       # Document templates
│   ├── resume/      # Resume templates
│   └── cover-letter/ # Cover letter templates
├── App.jsx
└── main.jsx
```

## Available Templates

- Modern

- Executive
- Creative
- Professional

- Tech

## Features

- Live preview
- Multiple templates for resumes and cover letters
- Save to localStorage
- PDF download (ready for integration)
